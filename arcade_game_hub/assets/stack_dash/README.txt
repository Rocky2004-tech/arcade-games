Stack Dash Asset Directory

This directory should contain the following assets:

1. characters/
   - player_idle.png
   - player_run_spritesheet.png
   - player_jump.png

2. tiles/
   - tile_blue.png
   - tile_red.png
   - tile_shadow.png

3. backgrounds/
   - parallax_layer1.png
   - parallax_layer2.png
   - parallax_layer3.png

4. sounds/
   - jump.wav
   - pickup.wav
   - drop.wav
   - fall.wav
   - powerup.wav
   - success.wav

5. music/
   - main_theme.mp3
   - lava_theme.mp3

Note: The game will create placeholder graphics if these assets are not found.
